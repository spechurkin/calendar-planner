# 📅 Calendar Planner
## 🇺🇸 English version

### About the Project

**Calendar Planner** is a simple and convenient web application designed to help groups of people quickly find common free time for meetings, trainings, rehearsals, trips, and other events.

Instead of endless chats like "who's free when?", each participant simply marks their **busy** days in the calendar. The system automatically shows when everyone is free and who is busy on a particular day.

### Key Features

- Personal calendar for each user
- Multiple projects/groups (team, sports club, friends, etc.)
- Clear visualization of availability
- Automatic search for the nearest common free dates
- User registration and authentication
- Russian and English language support
- Clean and intuitive interface

### Who is it for?

- Work and project teams
- Sports teams and clubs
- Creative groups (bands, theater, dance)
- Groups of friends
- Any community that needs to coordinate schedules regularly

## 🇷🇺 Русская версия

### О проекте

**Calendar Planner** — это удобное веб-приложение, которое помогает группам людей быстро находить общее свободное время для встреч, тренировок, репетиций, поездок и любых других мероприятий.

Вместо долгих переписок «кто когда свободен» каждый участник просто отмечает в календаре свои **занятые** дни. Система автоматически показывает, в какие дни все свободны и кто именно занят в выбранный день.

### Основные возможности

- Персональный календарь для каждого участника
- Несколько проектов/групп (команда, спортивная секция, друзья и т.д.)
- Наглядная визуализация занятости
- Автоматический поиск ближайших общих свободных дат
- Регистрация и авторизация
- Поддержка русского и английского языков
- Простой и интуитивно понятный интерфейс

### Для кого подойдёт

- Рабочие и проектные команды
- Спортивные команды и клубы
- Творческие коллективы (музыка, театр, танцы)
- Компании друзей
- Любые группы, которым нужно регулярно согласовывать общее время
